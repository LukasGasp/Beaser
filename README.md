## Beaser
A little python/java project for using a Beamer as Laser over sACN

## DMX-Footprint
Channel 1:  Mode <br />
Channel 2:  Dimmer <br />
Channel 3:  PAN     (x-pos) <br />
Channel 4:  Tilt    (y-pos) <br />
Channel 5:  x-Size <br />
Channel 6:  y-Size <br />
Channel 7:  R <br />
Channel 8:  G <br />
Channel 9:  B <br />
Channel 10: Effekt options <br />
    - Image: <br />
    1: NGK-Logo <br />
    2: NGK (Written) <br />
    3: FMT <br />

## Modes: (Channel 1)
0: OFF <br />
1: Horizontal Line <br />
2: Vertical Line <br />
3: Rectangle <br />
4: Oval <br />
5: Poly Line <br />
6: Image <br />
7: Arc <br />
